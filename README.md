# Telegram JSON to Markdown Converter

Этот сервис позволяет конвертировать JSON файлы экспорта Telegram в формат Markdown (.md) с возможностью выбора количества обрабатываемых сообщений и выгрузки результата в ZIP архиве.

## Возможности

- Загрузка JSON файла экспорта Telegram
- Выбор количества обрабатываемых сообщений (все, 10, 50, 100, 500, 1000)
- Конвертация сообщений в формат Markdown
- Отображение прогресса обработки
- Выгрузка результата в ZIP архиве

## Требования

- Современный браузер с поддержкой JavaScript ES6+

## Использование

1. Откройте `index.html` в браузере
2. Выберите JSON файл экспорта Telegram
3. Укажите количество сообщений для обработки
4. Нажмите кнопку "Конвертировать в Markdown"
5. После завершения обработки нажмите "Скачать ZIP архив"

## Структура проекта

- `index.html` - основной HTML файл с интерфейсом
- `style.css` - стили оформления
- `script.js` - основная логика конвертации
- `README.md` - документация

## Особенности реализации

- Обработка различных типов сообщений (текст, упоминания, ссылки)
- Поддержка медиафайлов в сообщениях
- Прогресс-бар во время обработки
- Автоматическая генерация JSZip для создания архива

## Формат вывода

Каждое сообщение в Markdown файле содержит:
- Дату и время сообщения
- Имя отправителя
- Текст сообщения с обработкой специальных элементов (ссылки, упоминания)
- Информацию о медиафайлах (если есть)
